Login with signed challenge

Authenticate by signing the challenge with ECDSA P-256.

Request Body schema: application/json
required
username
required
string <= 255 characters

User email / login identifier

challenge_id
required
string <uuid>

Challenge ID from the previous step

challenge
required
string

The raw challenge plaintext from Step 1

signature
required
string

Base64-encoded ECDSA P-256 signature over the challenge bytes

public_key
required
string

Base64-encoded ECDSA P-256 public key (X || Y bytes, uncompressed). Required on first login to bind the device; ignored on subsequent logins.

required
object (DeviceInfo)
